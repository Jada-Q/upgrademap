# UpgradeMap — 一页 Pitch

---

## 一句话

**用真实成交数据发现东京正在升级的街区，在别人注意到之前。**

---

## 问题

在东京买房或投资时，区域选择是最关键的决策，但判断依据严重不足：

| 现有方式 | 问题 |
|----------|------|
| 听中介推荐 | 信息不对称，无法验证 |
| 看新闻/社群 | 滞后性强，等你看到时已经涨了 |
| 自己查数据 | 数据分散在日文政府网站，23区逐个查耗时巨大 |
| SUUMO / LIFULL | 看得到挂牌价，看不到趋势和信号 |

**核心痛点：没有一个工具能直接告诉你「哪个区正在升级、处于什么阶段」。**

---

## 解法

UpgradeMap 交叉分析两组官方数据，自动识别升级信号：

```
真实成交价（国土交通省）  ×  人口趋势（総務省）
         ↓                         ↓
    房价动量评分              人口活力评分
         ↓                         ↓
              → 综合升级评分 + 信号分类
```

**三种信号：**
- **早期升级**：价格低于均值但在涨 + 人口增长 → 买入窗口
- **活跃升级**：价格和人口双增 → 已在升级中
- **成熟/风险**：价格高位 + 人口放缓 → 注意回调

## 差异化

| | UpgradeMap | SUUMO/LIFULL | 中介推荐 |
|--|-----------|-------------|---------|
| 数据颗粒度 | 区级趋势分析 | 单套房源 | 口头经验 |
| 信号判断 | 自动分类 | 无 | 主观 |
| 数据来源 | 政府公开数据 | 卖方挂牌价 | 不透明 |
| 语言 | 中文 | 日文 | 看人 |
| 成本 | 免费 | 免费 | 隐性佣金 |

**核心壁垒：不是数据多，是把散落的政府数据变成可操作的判断信号。**

---

## 目标用户

1. **在日华人购房者** — 买房前筛选区域，最大群体
2. **华人不动产投资者** — 追踪多区动态，寻找价格洼地
3. **不动产从业者** — 向华人客户展示区域趋势时的数据工具

初始市场规模：在日华人 ~80 万人，其中有购房需求的估计 10-20 万人。

## 商业模式（探索中）

**Phase 1 — 免费工具获客**
- 免费开放所有基础功能
- 通过 SEO + 社群传播获取初始用户
- 收集反馈验证需求

**Phase 2 — 增值服务**
- 区域深度报告（PDF，付费下载）
- 自定义监控提醒（某区评分变化时通知）
- 与不动产中介合作导流

**Phase 3 — 数据服务**
- API 接入（面向不动产公司）
- 定制分析报告
